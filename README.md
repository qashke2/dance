# Радужный туннель — инструкция

## Что нужно сделать перед публикацией на GitHub Pages:

1. Положи в эту же папку (рядом с index.html) два файла:
   - `dance.png` — твой спрайтшит 2560x2560 (21 кадр по 512px)
   - `music.mp3` — твоя музыка

2. Спрайтшит должен быть раскладкой по строкам слева-направо, сверху-вниз:
   - 5 кадров в строке, 5 строк (25 ячеек), используются первые 21.
   - Если у тебя другая раскладка (например 7x3 или по-другому пустые ячейки),
     скажи мне точное расположение — поправлю COLS/TOTAL_FRAMES в коде.

3. Залей все 3 файла (index.html, dance.png, music.mp3) в репозиторий GitHub.

4. Включи GitHub Pages: Settings -> Pages -> Source: main branch / root.

## Как это работает
- Кнопка "НАЧАТЬ" запускает музыку и анимацию (нужно по требованиям браузеров —
  звук нельзя включить автоматически без клика).
- Туннель и блёстки рисуются на canvas, цвета и пульсация синхронизированы
  с басами трека через Web Audio API (analyser).
- Танцующий спрайт по центру меняет кадры, ускоряется в такт музыке.

## Если хочешь поправить:
- Размер персонажа: `canvas.height * 0.5` в функции `drawSprite`
- Скорость анимации спрайта: `FRAME_DURATION` (мс на кадр)
- Количество блёсток: `for (let i = 0; i < 80; i++)`
- Яркость пульсации туннеля: `1 + beat * 0.5` в `drawTunnel`
